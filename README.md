Welcome to **Sugar Clouds Academy** – Where Dessert Dreams Come True!

### 🔗 [Live Demo: View the Website Here](https://dareen-housam.github.io/sugar-clouds/)

## Overview
Sugar Clouds is a visually rich landing page for a dessert academy. It showcases courses, team members, student success stories, and an interactive application process—all wrapped in a sweet, modern, and fully responsive design.

## 🚀 Features
- **Responsive Layout:** Optimized for desktop, tablet, and mobile devices.
- **Smooth Navigation:** Links in the navbar scroll smoothly to each section.
- **Home Section:** Welcoming hero intro with interactive call-to-action buttons.
- **Courses & Curriculum:** Details on signature baking courses and skill elevation.
- **Featured Works:** A high-quality gallery of dessert creations.
- **What We Offer:** Service categories like Masterclasses, Catering, and Consulting.
- **Why Choose Us:** Highlights academy strengths with clean iconography.
- **Live Statistics:** Counters for successful events and satisfied clients.
- **Pricing Packages:** Elegant pricing cards for Essentials, Advanced, and Masterclass tiers.
- **Success Stories:** Real student testimonials with metadata and images.
- **CEO & Team:** Profiles of the visionary leaders and instructors.
- **Online Application:** A clean, functional form for potential students to enroll.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure for better SEO.
- **CSS3:** Custom styles, Flexbox, CSS Grid, and modern hover animations.
- **Font Awesome:** Professional iconography.
- **JavaScript:** For the mobile responsive menu toggle.

## 📁 Folder Structure
- `index.html` – The main website file.
- `css/` – Stylesheets (`style.css`, `all.css`).
- `photos/` – Optimized images and dessert photography.
- `webfonts/` – Font Awesome source fonts.
